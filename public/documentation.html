My Flix API - Documentation

Intro

The My Flix API is a RESTful web service that provides users access to a comprehensive database of movies, directors, and genres. This API was built using Node.js, Express, and MongoDB. It enables users to explore movie details, manage their profiles, and curate a list of their favorite movies. The API supports CRUD operations, ensuring seamless interaction with movie-related data, user authentication, and profile management. It was designed as part of a full-stack MERN application. The My Flix API is the backend foundation for delivering a dynamic and engaging movie discovery experience.

API Endpoints

Description Method Enpoint Parameter Request Body Response Body
Gets a list of all movies GET /movies None None Returns status code 200 and a JSON array of all movie objects
Gets information on specific movie GET /movies/:Title Movie name None Returns status code 200 and a JSON object of information for specified movie
Gets information on specified genre GET /movies/genre/:genreName Genre name None Returns status code 200 and a JSON object of information on the genre
Gets information on specified director GET /movies/directors/:directorName Director's name None Returns status code 200 and a JSON object of information for specified director
